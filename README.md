# Sports Hub

Live: https://sports-hub-3nv4.onrender.com/

A minimal React + TypeScript + Vite app.

## Getting Started (Windows)
- Install Node.js 18+ and npm
- Verify installation:
  ```powershell
  node -v
  npm -v
  ```
- Install dependencies:
  ```powershell
  npm install
  ```

## Run in Development
- Start the dev server:
  ```powershell
  npm run dev
  ```
- Open: http://localhost:5173

## Build for Production
- Create a production build:
  ```powershell
  npm run build
  ```
- Preview the built app locally:
  ```powershell
  npm run preview
  ```

## Notes
- If you change the port, run: `npm run dev -- --port 5174`
- If install fails, remove `node_modules` and try `npm install` again.
